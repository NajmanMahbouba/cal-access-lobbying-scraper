# CAL-ACCESS Lobbying Data Scraper
Python scraper for California CAL-ACCESS lobbying employer records

This project is a Python web scraper employing Selenium and BeautifulSoup to navigate California CAL-ACCESS lobbying employer records. It was built in the fall of 2023. The script opens employer pages, accesses activity/financial records, parses the page HTML, and searches for targeted lobbying-related values. 

## Tools Used
- Python
- Selenium
- BeautifulSoup

## Why
I was interested in how public lobbying data could be accessed and analyzed more efficiently for economic and social science research.
